# skilllab

<p align="center">
  <img src="./assets/readme/hero.jpg" width="100%" alt="Skilllab: audits, merges, renames, prunes, and publishes agent skills">
</p>

Skilllab is the maintenance toolkit for the OCAS skill library. It audits for orphans and broken frontmatter, merges overlapping skills into class-level umbrellas, renames and repositions skill directories, prunes stale entries, and publishes skills to agentskills.io.

**Capabilities:**
- Audit: orphans, broken frontmatter, author gaps
- Merge: consolidate overlapping skills
- Publish: push sanitized skills to agentskills.io
